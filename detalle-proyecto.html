Detalles de los Proyectos

Proyecto 1: Desarrollo de un prototipo de Hoja de Vida electrónica con HTML y CSS

Descripción: Este proyecto consistió en el desarrollo de un prototipo interactivo de Hoja de Vida electrónica utilizando HTML y CSS para crear la estructura y el diseño visual. El objetivo principal fue crear una presentación profesional y atractiva de la información personal y profesional.

Tecnologías Utilizadas:

  • HTML5
  • CSS3

Funcionalidades:

  • Diseño de la interfaz de usuario
  • Organización de la información de la Hoja de Vida

Imágenes:

Logros: Logré desarrollar un diseño atractivo y funcional que resalta la información importante de la Hoja de Vida, proporcionando una experiencia de usuario intuitiva.

Proyecto 2: Desarrollo de una lista de contactos utilizando JavaScript

Descripción: Este proyecto consistió en el desarrollo de una aplicación web para gestionar una lista de contactos. Se utilizó JavaScript para implementar la lógica de la aplicación y manipular el DOM. El objetivo principal fue crear una herramienta sencilla pero efectiva para gestionar contactos de manera eficiente.

Tecnologías Utilizadas:

  • HTML5
  • CSS3
  • JavaScript

Funcionalidades:

  • Agregar, eliminar y editar contactos
  • Buscar contactos por nombre o categoría

Imágenes:

Logros: Implementación de una interfaz de usuario interactiva y funcional para la gestión de contactos.
